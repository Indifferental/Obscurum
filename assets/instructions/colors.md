## [RU] Инструкция по установке собственного цвета
1. Откройте в поисковике Google палитру RGB-цветов и выберите на ней любой понравившийся цвет

<p align="center"><img src="https://github.com/user-attachments/assets/f46e880a-5657-4e84-ba25-2fd0f5991f2c" alt="logo" style="width: 800px"/></p>

2. Скопируйте появившийся снизу HEX-код (например, #fc8981). Рекомендуется использовать именно формат HEX, но есть возможность поставить цвет и через другой формат (например, rgb(252 137 129))
3. Нажмите на кнопку в меню с надписью "Свой цвет", в появившемся окне вставьте полученный код и нажмите OK

<p align="center"><img src="https://github.com/user-attachments/assets/bfad8773-4506-4937-a239-9bd0ebb04369" alt="logo" style="width: 500px"/></p>

4. Чтобы сбросить цвет, нажмите на кнопку "Свой цвет" и далее OK ничего не вставляя, цвет сбросится на ранее установленный акцентовый цвет

## [EN] Instructions for installing your own color

1. Open the RGB color palette in Google search and select any color you like

<p align="center"><img src="https://github.com/user-attachments/assets/f46e880a-5657-4e84-ba25-2fd0f5991f2c" alt="logo" style="width: 800px"/></p>

2. Copy the HEX code that appears below (e.g., #fc8981). It's recommended to use the HEX format, but you can also specify the color using another format (e.g., rgb(252 137 129))
3. Click the button in the menu labeled "Custom Color", paste the resulting code into the window that appears and click OK

<p align="center"><img src="https://github.com/user-attachments/assets/bfad8773-4506-4937-a239-9bd0ebb04369" alt="logo" style="width: 500px"/></p>

4. To reset the color, click the "Custom color" button and then OK without inserting anything. The color will reset to the previously set accent color
